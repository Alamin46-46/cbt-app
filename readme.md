The Mini CBT System is a web-based application developed using Python and Flask. It simulates a simple computer-based test environment where users can answer multiple-choice questions and receive their results instantly.

The system presents a set of questions with multiple options, allows users to select their answers, and calculates the final score upon submission. Each test attempt is timestamped using Python’s datetime module.

This project is built using Object-Oriented Programming (OOP) principles. A Question class is used to represent each question, while a Quiz class manages the list of questions, user answers, and score calculation.

How to Run the Project

Install Flask: pip install flask
Run the application: python app.py
Open your browser and go to: http://127.0.0.1:5000/
